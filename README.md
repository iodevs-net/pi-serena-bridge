# Omni-Pi: Universal Semantic Intelligence Bridge

**Omni-Pi** es el puente de inteligencia semántica proactiva para `oh-my-pi`. Conecta la potencia de **Serena MCP** con tu flujo de trabajo de codificación, eliminando el gap de contexto y previniendo errores de arquitectura en tiempo real.

## 🚀 Características Elite (v5.0)

- **Jerarquía de Compresión de 3 Capas**: Optimiza el presupuesto de tokens priorizando Exportaciones (`E>`) y Estructura (`S>`) sobre Implementación (`I>`).
- **Detección de Intento por Stemming**: Identifica la intención de refactorización o corrección de bugs en lenguaje natural y prepara el contexto automáticamente.
- **Análisis de Impacto Paralelo**: El Gatekeeper de seguridad valida el impacto de tus cambios en todo el proyecto de forma instantánea mediante procesamiento en paralelo.
- **Zero-Latency Design**: Timeouts robustos y manejo de errores silencioso para no interrumpir nunca el flujo creativo.

## 🛠️ Instalación y Uso

Omni-Pi se integra como una extensión nativa de `oh-my-pi`.

1. Asegúrate de tener **Serena MCP** instalado.
2. Clona este repositorio en tu carpeta de extensiones de `omp`.
3. ¡Listo! Omni-Pi se activará automáticamente al detectar proyectos indexados.
